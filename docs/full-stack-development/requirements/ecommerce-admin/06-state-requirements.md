# 电商管理后台 - 状态需求

## 全局状态

- Loading：页面骨架屏，表格区域用 HeroUI Skeleton。
- Empty：显示当前筛选下无结果，并提供清空筛选动作。
- Error：网络或服务错误用 Alert 展示，可重试。
- Success：保存或规则更新成功用 Toast。
- Disabled：权限不足或数据状态不允许操作时，按钮禁用并用 Tooltip 解释。
- Mobile：列表列数收缩，关键字段优先，详情改为全屏或底部优先的 Drawer。

## 库存批次预警状态

- 正常：剩余天数大于阈值，使用中性或成功语义。
- 临期：剩余天数小于等于阈值且未过期，使用 warning 语义。
- 过期：当前日期超过有效期，使用 danger 语义，并在表格中优先排序。
- 未配置有效期：显示异常标签，进入数据修复流程。

## 权限状态

- 可查看可操作：显示完整字段和操作。
- 可查看不可操作：字段可见，操作禁用并解释原因。
- 不可查看敏感字段：字段脱敏，详情区域显示权限提示。
- 完全无模块权限：页面级 Alert，导航项可隐藏或禁用，但路由仍必须后端校验。
